# AutoHub  
Full-Stack Car Marketplace & Dealership Management System

Live Application: Deployed on Render  
GitHub Repository: https://github.com/FrankieeGodwin/AutoHub

---

## Project Overview

AutoHub is a full-stack web application designed to digitalize the buying and selling of new and used cars. It provides a secure, transparent, and interactive online marketplace connecting Admins, Dealers, and Customers, each with clearly defined roles and access controls.

The platform automates traditional car-trading processes such as listings, communication, payments, and analytics within a unified system.

---

## User Roles

### Admin
- Manages dealers and customers
- Monitors listings and transactions
- Views analytics, reports, and platform activity
- Oversees system security and performance

### Dealer
- Lists new and used cars
- Bulk uploads cars using CSV
- Manages vehicle media and features
- Tracks customer interactions and sales

### Customer
- Browses and compares vehicles
- Filters cars by brand, price, mileage, fuel type, and condition
- Manages favorites and wishlist
- Communicates with sellers using real-time chat
- Makes secure payments to access seller details

## Architecture

The project follows the Model–View–Controller (MVC) architecture:

- Models define MongoDB schemas for users, cars, dealers, payments, chats, and notifications
- Controllers handle business logic, authentication, and API responses
- Views are built using React and consume REST APIs

Asynchronous operations are handled using async/await for better readability and reliability.
